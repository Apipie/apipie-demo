PUT /api/v2/users/:user_id/tweets/:id
Update a tweet

Examples

PUT /users/980190962/tweets/980190962
{
  "tweet": {
    "scheduled_at": "2012-06-27 17:55:59 UTC",
    "sent": false,
    "text": "MyText",
    "user_id": "980190962"
  }
}
